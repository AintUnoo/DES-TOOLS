Assignment Brief — What is DES?

The Data Encryption Standard (DES) is a symmetric-key block cipher standardised by NIST in 1977 (FIPS PUB 46). It operates on 64-bit data blocks using a 56-bit effective key (supplied as 64 bits with 8 ignored parity bits). The algorithm uses a Feistel network structure with 16 identical rounds, where the same f-function is applied repeatedly with different subkeys derived from the original key.

Although DES is now considered cryptographically weak due to its short key length (vulnerable to exhaustive brute-force search), it remains the canonical example for teaching classical symmetric encryption concepts: confusion via S-boxes, diffusion via permutations, and the key schedule mechanism.

Implementation Approach

Per the assignment constraint, no DES or cryptographic libraries were used. The entire cipher is implemented in Python running client-side via the Pyodide WebAssembly runtime. The codebase is organised into clearly separated modules: permutation tables, bit utilities, padding, key schedule, the Feistel function, block-level encrypt/decrypt, multi-block ECB handler, and the key generator.

Ciphertext is output as uppercase hexadecimal, and input text is UTF-8 encoded then PKCS#5 padded before encryption — ensuring correct handling of any text length.
